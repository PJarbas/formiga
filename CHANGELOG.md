# Changelog

## 0.1.0 (2026-05-02)

Initial release of Tamandua — the agent team orchestrator for pi.

### Features

- Three bundled workflows: feature-dev, security-audit, bug-fix
- Pi-based agent scheduling (polling + work execution)
- SQLite-backed run and step state tracking
- Dashboard web UI for monitoring runs
- Role-based agent configuration
- Story-based loop steps
- Automatic retry with escalation
- Event logging with webhook support
- Medic health check system
- ASCII anteater easter egg
